Toy API documentation:

Routes

All HTTP methods are supported. Base port: 3001.

  • Get all products: /toys
  • Get toy by name: /search/name
  • Get toy by category: /search/category
  • Get toy by info: /search/info

    • Add toy- (only for register user - must send user's auth-token through headers): /toys
  • name: String
  • info: String
  • price: Number
  • cat: String
  • img: String
  • date: { type: Date, default: Date.now() }
  • user_id: String

    • Edit product - (only for register user - must send user's auth-token through headers): /toys/:idEdit
  • name: String
  • info: String
  • price: Number
  • cat: String
  • img: String
  • date: { type: Date, default: Date.now() }
  • user_id: String

    • Delete product - (only for register user - must send user's auth-token through headers):/toys/:idDel
  • name: String
  • info: String
  • price: Number
  • cat: String
  • img: String
  • date: { type: Date, default: Date.now() }
  • user_id: String
